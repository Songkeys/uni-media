# Media List

Here is a list of media that can be detected and compatible in the `<uni-media>` component.

- Image: `image/*`
- Video: `video/*`
- Audio: `audio/*`
- 3D Model: `model/*`
- PDF: `application/pdf`
- HTML: `text/html`

Any other feature request or bug report is welcome at [GitHub](https://github.com/Songkeys/uni-media)
